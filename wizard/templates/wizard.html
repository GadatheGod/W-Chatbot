Deployment Mode

Choose how you want WebAI Chat to handle AI responses.
🖥️

Local

Runs entirely on your server using Ollama. No internet needed for AI responses.

  • 100% data privacy
  • No API costs
  • Works offline
  • Requires GPU (optional)
☁️

Cloud

Uses cloud APIs like OpenAI, Anthropic, Google. Best quality, requires API key.

  • Best model quality
  • No GPU needed
  • Fast responses
  • Per-token costs
🔀

Hybrid

Tries local model first, falls back to cloud if unavailable. Best of both worlds.

  • Cost optimization
  • High availability
  • Requires both Ollama + API key
  • Most complex setup
